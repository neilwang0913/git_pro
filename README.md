# git_pro

## View Branch List
To view the list of branches, use the following command:
```console
$ $ git branch xyz
```
## Create branch   :
```console
$ git branch xyz
```

## Delete branch   :
To delete the branch named xyz, use the following command:
```console
$ git branch -d xyz
```

## Merge branch   :
```console
$ git checkout main -> git branch x1y2z3 x3y2z1
```
