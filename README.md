# git_pro

## View Branch List
To view the list of branches, use the following command:
```console
$ git branch
```
## Create branch   :
```console
$ git branch xyz
```

## Delete branch   :
```console
$ git branch -d xyz
```

## Merger branch   :
```console
$ git checkout main -> git branch x1y2z3 x3y2z1
```
